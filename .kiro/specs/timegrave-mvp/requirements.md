# TimeGrave MVP 요구사항

## 프로젝트 개요

TimeGrave는 미래에 열리는 타임캡슐 서비스로, 신비롭고 영적인 경험을 제공합니다.

## 핵심 기능

### AC-001: 랜딩 페이지

사용자가 접속하면 신비로운 입구(The Gate)를 경험해야 합니다.

- 안개 효과와 영혼의 입자 애니메이션
- "ENTER GRAVE" 버튼으로 진입

### AC-002: 묘지 대시보드

사용자의 타임캡슐들을 비석 형태로 표시해야 합니다.

- 잠긴 비석과 열린 비석 구분
- 남은 날짜 표시
- 그리드 레이아웃

### AC-003: 타임캡슐 생성

새로운 기억을 묻는 의식을 제공해야 합니다.

- 크립텍스 스타일 날짜 선택
- 텍스트/이미지/음성 업로드
- 친구 초대 기능

### AC-004: 타임캡슐 열람

봉인이 해제된 타임캡슐을 볼 수 있어야 합니다.

- 부활 애니메이션
- AI 처리된 콘텐츠 표시
- 다운로드 기능

## 기술 스택

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- React

## 비기능 요구사항

- 반응형 디자인
- 부드러운 애니메이션 (60fps)
- 접근성 고려
